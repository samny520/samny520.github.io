前提准备

环境准备：

  • Windows7 32 位系统
    • Shellcode 使用 kali linux Metasploit 生成 shellcode

Windows7 需要安装的软件：

  • Python2.7
  • pip install pyinstaller
  • pip install pywin32
  • VCForPython27.msi(微软官网可以下载)
